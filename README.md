#  WeCode

**Stop Solving Alone. Start Solving Together.**

WeCode is a real-time collaborative coding platform built for DSA practice, pair programming, and interview preparation. Multiple users can join a room, write code together, chat, use a shared whiteboard, and run code instantly.

---

## Features

- Real-time collaborative code editor
- Room-based sessions
- Live presence indicators
- Integrated chat
- Collaborative whiteboard
- Multi-language execution (C++, Java, Python, JavaScript)
- Clean & responsive UI

---

## Tech Stack

**Frontend**
- Next.js
- TypeScript
- Tailwind CSS
- Socket.io Client

**Backend**
- Node.js
- Express
- Socket.io

**Code Execution**
- Judge0 Community Edition (Public API)
